Overview

The Pride Theme build using very light weight, Sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development.

Before we start, lets take a look on the folder structure of the theme. All theme files are placed in the HTML folder. The main HTML folder consists of more folders for diferent types of theme assets:

  1. images - folder contain all images for the theme design
  2. js - folder contain all jquery plugin
  3. fonts - Folder contain font files used in theme
  4. css - folder contain all CSS files for the theme.

Installation

This is an HTML theme and doesn't require installation in the traditional meaning of the word. You only need to place the content (HTML files and the folders) in your prefered web directory

Keep in mind that the folder structure and file/folder names are important for theme to work in correct order.

Explaining common styles

This section explains some common style used in theme.

Text alignment

You can set text alignment by using common class like
.text-center{text-align:center !important;}
.text-right{text-align:right !important;}
.text-left{text-align:left !important;}

Text italic

You can set text italic by using common class like
.text-italic{font-style:italic}

Font case

You can set Font case i.e. uppercase/lowercase/none by using common class like
.lowercase{text-transform: lowercase;}
.uppercase{text-transform: uppercase;}
.nocase{text-transform:none;}

position

You can set any component position i.e. left and right by using common class like
.float-left{float:left !important;}
.float-right{float:right !important;}

You can used position(relative/fixed/absolute/static) by using common class like
.relative{position:relative !important;}
.static{position:static !important;}
.absolute{position:absolute !important;}
.fixed{position:fixed !important;}

Color style

Theme have mostly two color font style i.e. black and white. You can specify font color scheme by using class name like .color-white{color:#fff !important;} and .color-black{color:#212121 !important;}

Background color style

Theme have mostly 3 background color style i.e. black, grey and white. You can specify background color scheme in any section or block or any component by using common class like
.white-section{background-color:#fff;}
.grey-section{background-color:#f8f8f8;}
.black-section{background-color:#212121;}

margin and padding

In some cases you have to set 0px margin or 0px padding in that cases you can use common class like
.nomargin{margin:0 !important} .nopadding{padding:0 !important;}

You can see further detail about common style for padding and margin in style.css file with commented like "4.0-Padding and Margin"

Center content

You can set any component center of the screen or center of the main wrapper by using commno class like
.valign { position: relative; top: 50%; transform: translateY(-50%); -webkit-transform: translateY(-50%); -moz-transform: translateY(-50%); -o-transform: translateY(-50%); -ms-transform: translateY(-50%); } And you can also set content in center position by using alternet class like
.table-content {position: relative;display: table;height: 100%;text-align: center;width: 100%;} .table-center-text {display: table-cell;height: 100%;vertical-align: middle;}

Button style

Theme have used mainly 3 diffenrent style of button with 4 size. i.e. black background, black border style, White background, White border style, Round black background, Round white background, Round black border and round white border.

You can set button background style by using common style like
.btn-border-white, .btn-border-black, .btn-bg-black, .btn-bg-white, .btn-radius and btn-size-0,btn-size-1,btn-size-2,btn-size-3,btn-size-4
